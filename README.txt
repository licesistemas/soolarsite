SITE SOOLARWIN

Arquivos principais:
- index.html: página principal
- style.css: visual completo do site
- app.js: menu mobile, animações, calculadora, formulário WhatsApp e galeria
- assets/: logo e imagens hiperrealistas

Como publicar:
1. Edite no app.js o número do WhatsApp em WHATSAPP_NUMBER.
2. Edite no index.html telefone, e-mail e textos de contato.
3. Suba todos os arquivos desta pasta para a hospedagem, GitHub Pages, Netlify, Vercel ou Firebase Hosting.

Observação:
Este site é estático. O formulário abre uma mensagem pronta no WhatsApp e não salva dados em banco.


Atualização: textos revisados para destacar área de atendimento em todo o Brasil e Paraguai.
